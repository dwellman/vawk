# Glossary

- PLAN/CODE/TESTS/NOTES: Required structured sections for code responses.
- RAG: Retrieval-augmented generation using snippets and learn.awk sources.
- Promotion: Turning a chat turn into a runnable AWK job (`script.awk` + `spec.yaml`).
- Golden path: Documented end-to-end flow with commands, expected output, and receipts.
- POSIX AWK: Standards-compliant awk without gawk-specific extensions.
- Receipt: Logged output or JSON record that proves a run/test.
